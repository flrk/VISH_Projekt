
Projekt Name: Evolution of Airbnb over time

Team Members: Lennard Rössig, Florian Rückauf

Data Source: http://tomslee.net/airbnb-data-collection-get-the-data

d3 lib Source: https://cdnjs.cloudflare.com/ajax/libs/d3/4.2.2/d3.min.js
d3 code for collision with svg frame: https://bl.ocks.org/cmgiven/547658968d365bcc324f3e62e175709b

++Usage++
projekt can be opened on local maschine
launch directory in local http server
open browser with localhost:8080

++User Interaction++
drag slider to see changes over time
Buttons to select values to be seen as average, maximum or minimum
change scaling factor to absolute or relative

++Used visualization techniques++
attribute encoding through Size/Scale
colormapping for cities and tracability
